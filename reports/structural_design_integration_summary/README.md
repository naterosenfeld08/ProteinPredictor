# Structural Design Integration Summary

This folder contains a compact report of findings from structural benchmark integration work (WT/mutant discovery reliability + real ColabFold pilot metrics).

## Contents

- `findings_summary.txt`  
  Plain-text narrative summary suitable for email or quick sharing.

- `data/discovery_reliability_snapshot.json`  
  Snapshot of discovery fallback reliability on a realistic RCSB sample.

- `data/real_colabfold_pilot_metrics.json`  
  One-case real pilot metrics (`2OV0:A -> 2QDV:A`, mutation `M51A`) including baseline context.

- `data/key_metrics_table.csv`  
  Tabular metrics summary for quick plotting/reporting.
